# 1.1.0

## 变更

1. 调整目标解析逻辑，“一键避让焊盘”“统一文字方向”“设置字体”“设置固定字号”仅处理选中丝印或最近一次由扩展生成的丝印，不再默认回退到当前 PCB 全部丝印
2. 优化网络丝印生成策略，同一器件内相同网络在同一丝印层只生成一份文字，减少重复 `GND`、电源网等标签
3. 优化焊盘避让流程，新增丝印文字之间的互相避让，并通过空间索引减少大板子上的碰撞检测开销
4. 复用器件焊盘缓存参与候选计算，减少重复读取器件引脚列表

# 1.0.2

## 新增

1. 新增“统一文字方向”功能，自动将顶层丝印统一为 `0°`、底层丝印统一为 `180°`
2. 新增丝印字体设置与固定字号设置，并保存为扩展用户配置

## 变更

1. 更新“丝印生成器”，扩展名称、描述、关键词、菜单结构与关于信息
2. 菜单入口调整为仅在 PCB 页面提供，并补充“生成网络丝印”“一键避让焊盘”“统一文字方向”“设置字体”“设置固定字号”等命令

# 1.0.1 
## 新增
1. 新增“生成网络丝印”功能，可根据选中器件、器件焊盘或游离焊盘的网络名自动创建丝印文字
2. 新增丝印焊盘避让能力，支持优先处理选中丝印、最近一次生成的丝印，或当前 PCB 中全部丝印
